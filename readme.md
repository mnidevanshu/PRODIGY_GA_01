## Task 1 : Text Generation with GPT-2
This repository contains the implementation of a text generation tool using the GPT-2 Transformer model.

### 🚀 Project Overview
The application leverages OpenAI's GPT-2 to generate coherent, contextually relevant text from user-provided prompts. It features a custom Graphical User Interface (GUI) for seamless interaction with the model.

### Key Features
 * **Transformer-Based Generation**: Uses the Hugging Face transformers library for high-quality text output.
 * **Interactive GUI**: A user-friendly interface built with Tkinter for entering prompts and viewing results.
 * **Real-time Processing**: Fast tokenization and generation using PyTorch.
 * **Training Simulation**: Includes a simulated workflow for model fine-tuning.
   
### 🛠️ Technical Stack
 * **Language**: Python 3.x.
 * **Libraries**: Transformers, Torch.
 * **GUI Framework**: Tkinter.
