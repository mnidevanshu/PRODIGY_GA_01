## 📂 Task-01: Text Generation with GPT-2

### Description
This project focuses on fine-tuning the **GPT-2 (Generative Pre-trained Transformer 2)** model to generate coherent and contextually relevant text. It features an advanced GUI that allows users to train the model on custom datasets and generate text based on specific prompts.

### Features
 * **Model Fine-tuning:** Ability to train GPT-2 on custom text data to mimic specific styles.
 * **Interactive GUI:** A clean interface for inputting prompts and viewing real-time generation.
 * **Custom Prompting:** Generate text blocks based on user-defined starting strings (e.g., "Artificial Intelligence").
